Objetivo da Aplicação:

1 - pegar dados populacionais dos paises de um arquivo externo CSV;

2- ler os dados do arquivo em uma coleção

3 - mostrar na tela os dados recebidos


Files:
Program.cs = executar a logica geral
Country.cs = classe que representa um unico país
CsvReader.cs = classe responsavel por importar os dados dos países do arquivo CSV e preencher minha coleção
populationFormatter.cs = classe responsavel por formatar dados da população de cada país (codigo do prof.|  não utilizado)
